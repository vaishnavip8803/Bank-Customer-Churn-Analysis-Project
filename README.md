# Bank-Customer-Churn-Analysis-Project

## Project Overview
This project analyzes bank customer data to understand why customers leave the bank. The project includes data cleaning, exploratory data analysis (EDA), and visualizations to find the factors that affect customer churn.

## Problem Statement
Banks lose customers every year, which affects their business and revenue. The goal of this project is to analyze customer data, identify the reasons for customer churn, and provide useful insights that help improve customer retention.

## Objectives
- Understand customer behavior.
- Clean and prepare the dataset.
- Perform Exploratory Data Analysis (EDA).
- Find the factors affecting customer churn.
- Provide business insights to reduce customer churn.

##  Dataset
The dataset contains information about bank customers, such as:
- Customer ID
- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Has Credit Card
- Is Active Member
- Estimated Salary
- Churn Status
